v4.0.32 - Notes de version

Version 4.0.32

  • Date de lancement : 2019-11-14
  • Téléchargements

Quoi de neuf?

  • Styles et plugiciels
    • Minor - Added some Bootstrap 4 helpers classes for spacing, along with stretched-link used to expand the click-able area of an hyperlink.
    • Minor - Share widget - Added rel="noreferrer noopener" attribute to the share buttons for cross-origin concerns
    • Minor - Tables - Added "scroll to top" effect as default behaviour when buttons are clicked in pagination
    • Minor - Data inview - Added "in"-only animations by using the in-only class
    • Patch - Data inview - Removed aria-hidden attribute from inview and outview content
    • Patch - Tables - Made table re-filter from all results after filtering once
    • Patch - Tables - Changed i18n label text for tables' pages navigation
    • Patch - Overlay - Removed duplicate word "Close" from the close button title
    • Patch - Overlay & Lightbox - Fixed autogenerated close button undefined error
    • Patch - Created IE polyfill for dashed underlines on <abbr>
  • Entretien
    • Highlight - Added CDTS address to the WET-BOEW's documentation home page
    • Dependencies - Versions updated for some e.g. Bootstrap
    • Build script - Validation errors fixed and/or enhanced e.g. Bootlint
